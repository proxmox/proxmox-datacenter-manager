===========
remotes.cfg
===========

Description
===========

The file ``/etc/proxmox-datacenter-manager/remotes.cfg`` is a configuration
file for Proxmox Datacenter Manager. It contains all Proxmox VE and Proxmox
Backup Server remotes that the PDM instances manages.
